Special Topics In Software Engineering

The failure of a presentation on a website can have a negative impact on quality of the website, the services it provides, and branding the company is trying to achieve. Presentation failure can easily occur in modern web applications due to the complex and powerful nature of HTML, CSS. Most of the companies follow specific font and colours for their website. They maintain consistency in their design throughout the website. Any anomaly in website CSS with consistency will lead presentation failure.

Intro About Our Project

As a part of UI development unit testing is an important task to check which things are visually missing according to design or website consistency. It is a very difficult task to testing team to check with every line on static pages to find the missing CSS and automating the process of CSS testing will be a useful tool to give a report on website CSS.

Previous Work

Literature review: (1) detection – identify presentation failures by comparing test web page rendering and its oracle, (2) localization – identify potentially faulty HTML and CSS elements responsible for the observed presentation failures in phase 1, and (3) result set processing – prioritize the set of potentially faulty HTML elements identified in phase 2. The output of phase 3 is reported to the user.
